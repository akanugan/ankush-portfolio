This portfolio is built with React 19 and Vite, a modern frontend stack optimized for performance. The design uses Tailwind CSS 4 for styling, Framer Motion for smooth animations, and shadcn/ui for accessible components. The site is deployed to GitHub Pages via an automated GitHub Actions workflow that rebuilds and redeploys on every push to the main branch. All content is statically generated, making it fast, secure, and easy to host. The dark theme with gold and electric blue accents is inspired by particle physics, featuring animated particle effects and horizontal sliding panels for Experience, Projects, and Publications sections.
